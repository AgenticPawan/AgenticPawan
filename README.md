## 🚀 Featured Project — AI.Copilot.Scaffolding

<div align="center">

[![Readme Card](https://github-readme-stats.vercel.app/api/pin/?username=AgenticPawan&repo=AI.Copilot&theme=tokyonight&border_color=6366f1&title_color=38bdf8&icon_color=14b8a6&show_owner=true)](https://github.com/AgenticPawan/AI.Copilot)

</div>

**`AI.Copilot.Scaffolding`** is a `dotnet new` template that instantly bootstraps a **production-ready GitHub Copilot workspace** into any project — .NET, Angular, Node.js, or empty folder.

```bash
# Install once
dotnet new install AI.Copilot.Scaffolding::2.0.1

# Scaffold into any project
dotnet new copilot-scaffold --projectName MyApp --dotnetVersion 10 --dbProvider PostgreSQL
```

**What gets scaffolded in one command:**

| Layer | Contents |
|---|---|
| 🤖 **Agents** | backend-architect, devops-engineer, frontend-engineer, fullstack-architect, qa-engineer |
| 📋 **Instructions** | backend, frontend, security, testing, multi-tenancy |
| 💬 **Prompts** | 15+ workflow prompts for features, bugs, reviews, tests |
| 🧠 **Skills** | Per-role SKILL.md files for structured Copilot behavior |
| 🔌 **MCP Server** | Bundled DotnetMcpServer — ready to connect |
| ⚙️ **CI/CD** | GitHub Actions scaffold compliance workflow |

> ✅ Framework-agnostic · Zero config · Non-destructive · VS Code ready

## 🧠 Copilot Workspace Architecture

The scaffold creates this structure **automatically in one command**:

```
your-project/
├── .github/
│   ├── agents/              ← 5 specialist AI agents
│   │   ├── backend-architect.agent.md
│   │   ├── devops-engineer.agent.md
│   │   ├── frontend-engineer.agent.md
│   │   ├── fullstack-architect.agent.md
│   │   └── qa-engineer.agent.md
│   ├── instructions/        ← 6 context-aware instruction files
│   ├── prompts/             ← 15+ workflow prompts
│   ├── skills/              ← Per-role SKILL.md files
│   ├── mcp-servers/         ← DotnetMcpServer (ready to connect)
│   └── copilot-instructions.md
├── .copilot-chat/
│   ├── 01-behaviour-layer/  ← Architecture + coding standards
│   ├── 02-task-layer/       ← Per-task AI guidance
│   ├── 03-persona-layer/    ← AI personas
│   └── 04-workflow-layer/   ← End-to-end workflows
└── .vscode/
    ├── mcp.json             ← MCP server config
    └── settings.json        ← Copilot settings
```
